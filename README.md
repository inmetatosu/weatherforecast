# Opti-Opal TypeScript Tool Reference

This repository serves as a comprehensive reference and starter project for building custom tools for Optimizely Opal using TypeScript and the Opal SDK. It provides a foundational structure and examples to help developers quickly get started with creating powerful, integrated solutions within the Optimizely ecosystem.

## üöÄ Getting Started

This project is designed to be a reference. To use it as a basis for your own Opal tools:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/egandalf/opti-opal-typescript-tool-reference.git
   cd opti-opal-typescript-tool-reference
   ```
2. **Install dependencies:**

   ```bash
   npm install
   # or
   yarn install
   ```
3. **Review the** `src/` **directory:** This is where the core logic for your Opal tools will reside. Explore the examples provided to understand how to interact with the Opal SDK.
4. **Configure your environment:** Refer to the `.env.example` file for necessary environment variables.
5. **Build your custom tool:** Extend the provided structure to implement your specific tool's functionality.
